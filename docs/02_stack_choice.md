# Выбор стека

## Python 3.11+

Python удобен для быстрой разработки прототипов, работы с аудио, serial-портами,
сетевыми адаптерами и Raspberry Pi.

## Vosk

Vosk выбран как основной backend распознавания, потому что работает офлайн и подходит
для потоковой обработки коротких голосовых команд.

## sounddevice

sounddevice используется для захвата аудио с микрофона. Он прост в применении и работает
поверх PortAudio.

## Typer

Typer даёт понятный CLI для запуска демо, проверки конфига и тестирования текстовых
команд без микрофона.

## Pydantic и YAML

Pydantic проверяет структуру конфигурации, YAML делает список фраз и параметры backend-а
удобными для редактирования.

## pytest, ruff, mypy

pytest проверяет бизнес-логику, ruff поддерживает единый стиль, mypy помогает находить
ошибки типов в интерфейсах.
