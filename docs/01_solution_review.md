# Обзор решений для распознавания речи

## Vosk

Vosk работает офлайн, поддерживает русский язык и запускается на слабом железе без GPU.
Подходит для Raspberry Pi и задач с ограниченным набором команд.

## Whisper

Whisper даёт хорошее качество распознавания, но даже малые модели могут иметь заметную
задержку на Raspberry Pi. Для данного проекта это избыточный вариант.

## Faster-Whisper

Faster-Whisper оптимизирует Whisper, но всё равно требует больше ресурсов, чем Vosk.
На ПК это хороший вариант, но для учебного робота на Raspberry Pi стек сложнее.

## PocketSphinx

PocketSphinx лёгкий и офлайн, но качество распознавания русского языка обычно ниже, а
настройка словарей менее удобна.

## Онлайн STT-сервисы

Онлайн-сервисы могут давать высокое качество, но требуют стабильного интернета и создают
зависимость от внешнего API.

## Вывод

Для Raspberry Pi выбран Vosk: он работает локально, не требует GPU, поддерживает русский
язык и достаточно прост для прототипа голосового управления роботом.
